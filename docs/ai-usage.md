This page is meant to describe the AI usage policy of this repository. This page is split up into a few sections
for different cases.

## Coding Agents Usage
Use of coding agents such as Claude, Copilot, etc. are prohibited. This is due to clause 4 of the BSD-3 license.
Coding agents pull in the repository and may upload the files of this repository into their dataset. That would be
a violation of the license.

## Copilot or other AI Agent PR Reviews
Again, use of AI agents to review code is prohibited. This is due to clause 4 of the BSD-3 license.
Coding agents pull in the repository and may upload the files of this repository into their dataset. That would be
a violation of the license.

## Copying and Pasting Large Chunks of Code into an Agent or Chatbot
Copying and pasting large chunks of code into a coding agent or some other chatbot is prohibited. This is due to clause 
4 of the BSD-3 license. These interactions are saved and used for improvement of their models.

## Small Snippets or Error Messages
Using a coding agent or chatbot as an advanced search engine is fine. Including small snippets to help figure out why
something is failing or breaking is fine.
